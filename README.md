# Smart Hospital Queue Management System

This project is an AI-powered hospital queue management system that prioritizes patients based on urgency.

## Features
- Patient Appointment UI
- NLP Complaint Analyzer
- Queue Prediction Model
- Crowd Detection using YOLOv8
- Hospital Flow Dashboard

## Technologies Used
- Python
- Flask
- SQLite
- OpenCV
- YOLOv8
- HTML
- CSS

## Modules
1. Crowd Detection
2. NLP Complaint Analyzer
3. Queue Prediction Model
4. Hospital Flow Dashboard
5. Patient Appointment UI

## How to Run
1. Install dependencies
2. Run `python app.py`
3. Open browser and go to `http://127.0.0.1:5000`
